This is my style tester page. I took inspiration from a few sources:

  • 9x Standard: Classic Windows 95/98 window decorator style
  • 9x Plum: A mix of the 9x Plum color scheme and After Dark's Flying Toasters screensaver (as a wallpaper)
  • 9x Matrix: High contrast 9x theme inspired by The Matrix. Also a theme I actually used when I was young, somehow !
  • 9x Mystery: Iconic theme from the Microsoft Plus! for Windows 95 software suite
  • 9x Standard: Classic Windows 95/98 window decorator style
  • XP Luna: Default theme (except when the RNG likes you !), based on Windows XP Luna Blue
  • XP Olive: Technically it should be XP Luna Olive
  • XP Silver: Same as above but in metallic silver
  • XP Royale Noir: Popular unreleased theme for Windows XP. Eventually became XP Zune in 2006
  • Whistler Watercolor: (Loosely) based on the scrapped Watercolor theme from pre-release versions of Windows XP
  • Common Desktop Environment: The only *NIX theme of the list, based on the eponymous CDE, itself based on (Open)Motif

You can enable the style selector on any page by entering the Konami Code (up, up, down, down, left, right, left, right, B then A).
