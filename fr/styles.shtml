Voici la page que j'utilise pour tester mes palettes graphiques. Je me suis inspiré de quelques styles rétro:

  • 9x Standard: Style classique de Windows 95/98
  • 9x Plum: Un mix du thème Plum de Windows 95 et de l'économiseur d'écran Flying Toasters d'After Dark
  • 9x Matrix: Thème à (très) haut contraste inspiré par Matrix
  • 9x Mystery: Thème iconique de Microsoft Plus! 95
  • XP Classic: Thème "de secours" sur Windows XP
  • XP Luna: Thème par défaut, basé sur Luna Blue de Windows XP
  • XP Olive: Techniquement je devrais l'appeler "XP Luna Olive"
  • XP Silver: Idem mais en gris métallisé
  • XP Royale Noir: Thème populaire jamais sorti officiellement, qui deviendra par la suite XP Zune en 2006
  • Whistler Watercolor: Basé sur le thème éponyme de Windows Whistler
  • Common Desktop Environment: Le seul thème d'origine *NIX, basé sur CDE, lui même basé sur (Open)Motif

Tu peux afficher le sélécteur de thèmes sur n'importe quelle page en rentrant le Konami Code !
